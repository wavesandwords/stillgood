# StillGood — 食品保质期查询工具

> 快速查询任何食品开封后是否还能继续使用。

## 功能

- 🔍 搜索查询：输入食品名称，即时返回保质期数据
- 📂 分类浏览：按分类快速查找
- ⏱️ 状态提示：新鲜 / 尽快使用 / 临近过期
- 📱 简洁界面：无需注册，打开即用

## 技术栈

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS**
- **TypeScript**

## 本地运行

```bash
npm install
npm run dev
```

打开 http://localhost:3000

## 数据来源

保质期数据综合整理自：
- 各国食品药品监督管理局指南
- 品牌官方说明
- USDA 食品安全指南
